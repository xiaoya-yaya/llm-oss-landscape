## 🔥 趋势洞察

**Coding Agent 赛道持续爆发，工具链走向一体化。** 本周 28 个新项目中，Coding Agent 以 21 个项目的绝对数量占据首位。以 [badlogic/pi-mono](https://github.com/badlogic/pi-mono)（⭐41.4k，OpenRank 增长 943%）为代表，新一代 coding agent 不再满足于单一的代码补全，而是将 CLI、LLM API 网关、TUI/Web UI、Slack 集成打包为一站式工具包。[ultraworkers/claw-code](https://github.com/ultrawers/claw-code)（⭐188.8k）和 [affaan-m/everything-claude-code](https://github.com/affaan-m/everything-claude-code)（⭐168.2k）虽然高星但争议不断，反映社区对 coding agent 的定义边界仍在博弈。

**知识图谱 + Graph RAG 成为新热点。** [abhigyanpatwari/GitNexus](https://github.com/abhigyanpatwari/GitNexus)（⭐31.0k，OpenRank 增长 1202%，本周增速之王）以纯浏览器端的知识图谱构建切入代码理解赛道，而 [tirth8205/code-review-graph](https://github.com/tirth8205/code-review-graph) 则将知识图谱用于 Claude Code 的代码审查。GraphRAG 分类下已有 4 个新项目，结合知识图谱的 RAG 方案正在从实验走向工程落地。

**AI Agent 基础设施日趋成熟，安全与沙箱受到重视。** [TencentCloud/CubeSandbox](https://github.com/TencentCloud/CubeSandbox) 为 AI Agent 提供轻量级并发沙箱，[KeygraphHQ/shannon](https://github.com/KeygraphHQ/shannon)（⭐40.5k，OpenRank 增长 385%）则用白盒方式自动化 Web 安全渗透测试。与此同时，MCP 生态继续扩张（12 个新项目），Context 窗口优化（[mksglu/context-mode](https://github.com/mksglu/context-mode)）和 Agent 记忆系统也在同步演进，表明 Agent 基础设施的各个层——工具协议、上下文管理、安全隔离——正在全面补齐。
